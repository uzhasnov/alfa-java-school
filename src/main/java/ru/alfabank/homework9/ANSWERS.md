1.  Нужно ли было впринципе писать свой собственный генератор с фамилиями? Было бы проще использовать генератор из datafaker?
    Писать собственный генератор с фамилиями нужно только если есть необходимость в какой-то специфичной логике работы, 
    например, если нужно выбирать из списка заранее заданных фамилий. В общем случае удобнее использовать datafaker.

2.  Какая из двух библиотек из этого задания может больше пригодится при написании автотестов - jfiglet или datafaker?
    При написании автотестов больше пригодится библиотека datafaker, т.к. в автотестах часто есть необходимость в 
    генерации тестовых данных.

3.  Каким образом JAR-файлы этих библиотек попали к нам в проект?
    Мы прописали необходимые зависимости в build.gradle файл, дальше gradle сам подтянул необходимые JAR-ники в проект.
